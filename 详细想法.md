EcoBrain
将内容从“很散”结构化为五个层级
按 数据层（输入） → 指标层（输出） → 分析层（报告模块） → 多智能体层（中间环节） → 架构层（UI/接口） 组织，便于落库、复算、产出报告与产品化。

建议：每条数据带来源/时间/空间/精度/许可/置信度/版本
建议：每个指标可追溯到数据依赖与计算口径
建议：每个 Agent 输出结构化中间结果（可审计）

1) 数据层（输入）：全量数据清单
建议为每条数据记录 来源、采集时间/有效期、空间范围、分辨率/精度、许可与合规、置信度、版本号。
可落库
复制本节文本
园区边界：园区红线、分区（工业/仓储/生活/绿地）、地块（宗地）
行政区边界：省/市/区县/街道（用于政策、电价、排放因子匹配）
地形与高程：DEM、坡度（光伏布置、管网、土建成本影响）
土地利用/覆盖：建设用地、水体、林地、耕地（可建设约束）
建筑物：轮廓、层数/高度（屋顶光伏、负荷估计 proxy）
道路/铁路/水运：物流成本、危化品运输约束
企业清单（POI/工商）：名称、统一社会信用代码（如可得）、坐标、地址、园区归属
行业分类：国标行业代码/NAICS 映射、细分到工艺（如：氯碱、合成氨、发酵）
规模与产能 proxy：员工、占地/建筑面积、设备规模线索、年报摘要（可选）
上下游关系（若可得）：主要产品/原料（用于工业共生与副产品消纳）
电力：变电站/线路/接入点位置与容量、可靠性指标（若可得）、分时/需量电价规则、电网排放因子（地区/时段/边际优先）
天然气/燃料：管网、调压站、可接入点与容量、气价（季节/阶梯）；煤/油/生物质/工业副产气（如焦炉气等）
热力与蒸汽：供热管网、热源厂、蒸汽等级（压力/温度）、价格与计量方式
水系统：取水水源、水价、用水指标、再生水供给、污水厂容量与出水标准
固废/危废：处置能力、处置价格、运输半径、合规资质
气象数据：逐小时/典型年（温度、湿度、风速风向、辐照、降水）
负荷气候指标：HDD/CDD、极端高温/严寒统计（韧性）
可再生资源：光伏（屋顶/地面可用面积、遮挡、辐照）、风电（风资源、限高/禁建）、生物质（可获得量/半径/价格）、地热/余热（如有）
碳汇与生态：绿地、水体等（如涉及园区碳中和叙事/项目）
能耗计量：电（小时/日/月曲线、最大需量、功率因数）、燃气/燃料（用量、热值、设备效率）、蒸汽/热/冷（用量、温压等级、回水温度）
生产数据：产量、开工率、班次（用于单位能耗反推）
排放数据：排污许可与在线监测（烟气流量、CO₂/NOx/SO₂/颗粒物），温室气体核算口径（S1/S2，必要时扩展 S3）
排放因子库：电力排放因子（区域/时段），燃料排放因子（含低位热值、碳含量、氧化率）
行业工艺模板：输入（原料/电/蒸汽/燃料/冷却水等）→输出（产品/副产品/废热品位/CO₂浓度等），单位能耗/物耗区间（按规模/地区修正）
技术措施库：适用条件、效率/性能曲线、寿命、CAPEX/OPEX、运维要求（含三联供、吸收式制冷、锅炉/窑炉、热泵、储能等）
CCU/捕集参数：捕集率、能耗、溶剂消耗、纯度要求、压缩/液化能耗
成本与价格库：设备造价、土建安装系数、地区造价指数；副产品/CO₂利用产品价格、运输成本、管网单位造价
政策与标准：地方能耗双控/碳达峰方案、园区/行业准入、补贴与试点
碳市场：配额规则、履约价格、CCER/自愿减排价格（如适用）
金融：贴现率/WACC、贷款利率、税率、折旧、补贴/返还机制
环评约束：排放总量控制、危化品管理、噪声、取水许可
建设约束：红线、管廊规划、消防间距、防爆区划
供能可靠性要求：重要负荷等级、备用容量要求
2) 指标层（输出）：指标全集
建议按 基线、方案结果、经济性、资源约束、风险与不确定性 五类组织；支持园区/企业/子区域分解。

产业结构：行业占比（企业数/产值 proxy/用能占比）
能源：年用电量、最大需量、年燃气/燃料用量、蒸汽/热/冷需求
能效：单位产量能耗（若可估）、综合能耗（tce）、能效对标分位数
碳排：S1、S2、合计；碳强度（tCO₂/产值或 tCO₂/产量）
污染物/资源：用水量、污水量、固废/危废量（若数据具备）
年节能量（电/热/燃料）
年减排量（tCO₂e/年，分 S1/S2）
可再生替代比例（%）
CO₂ 捕集量/利用量/封存量（t/年）、捕集率（%）
三联供：装机容量（MW）、供电/供热/供冷量、等效一次能效、燃气消耗
余热回收：回收热量（GJ/年）、温度品位分布
微网与储能：储能容量（MWh）、削峰填谷量、弃光率、供电可靠性提升（如可量化）
能量平衡：8760 小时能量流（可选输出典型日）
副产品清单：种类、产量（t/年）、品质指标（纯度/温度/压力等）
消纳匹配：潜在下游数量、平均距离、可实现率（基于约束评分）
经济价值：副产品销售收入、纯化/压缩/储运成本、净利润贡献与价格敏感性
协同减排：替代外购能源/原料带来的间接减排（可单列）
CAPEX、OPEX、年度现金流
NPV、IRR、静/动态回收期
LCOE/LCOH/LCOC（电/氢/冷等）
边际减排成本 MAC（¥/tCO₂）
资金需求曲线与分期建设建议（现金流可承受性）
光伏可开发容量（MWp）与利用小时
风电可开发容量（如适用）
生物质可获得量（t/年）与上限成本
用水约束占用（m³/年）、排水许可占用
电网/气网/热网接入裕度（容量占用%）
关键参数敏感性（电价/气价/碳价/开工率/融资成本/副产品价格）
场景对比：保守/中性/激进（或政策情景）
置信度：数据完备度评分、结论可信等级（A/B/C）
3) 分析层（报告模块）：固定章节与分析角度
建议固定为 12 个模块，保证报告可复用、可审计、可自动生成；附录强制包含参数表与引用。
可模板化
复制本节文本
执行摘要：结论、Top 3 方案、投资与收益、减排与达标路径
园区范围与数据说明：选区边界、数据来源、缺口、口径（S1/S2）
产业与空间画像：行业分布、重点企业、产业链特征、用地结构
能源现状与负荷画像：电/热/冷/蒸汽、峰谷、季节性、关键约束
碳排基线核算：方法、排放因子、结果与区间、主要排放源贡献
资源禀赋与外部条件：风光、生物质、水、电网气网、气候（HDD/CDD）
机会清单（技术筛选）：措施库匹配、适用性评分、必需补数
园区供能系统方案：三联供/热泵/储能/绿电微网组合与运行策略
CCU/CCUS 与 CO₂ 源汇匹配：来源画像、捕集路线、利用/封存路径比较
工业共生与副产品价值：供需网络、物流/管网、可变现量与收入模型
经济性与投融资：CAPEX/OPEX、现金流、NPV/IRR、MACC、分期路线图
风险、敏感性与实施计划：技术/市场/合规/数据风险、情景结果、里程碑与补数计划
附录（强制）：参数表、公式与模型版本、引用清单、结果明细表（可导出 Excel）。
4) 多智能体层：从选区到报告的流水线
每个环节产出 结构化中间结果（可落库、可追溯、可复算）。建议统一“共享黑板”字段。
需要编排
复制本节文本
区域解析 Agent（Geo Resolver）：polygon/行政区/手绘范围 → 空间统计、企业/设施清单、图层裁剪、数据完备度
企业匹配与去重 Agent（Entity Resolver）：POI↔工商↔园区名录合并 → entity_master（含置信度与证据）
行业与工艺映射 Agent（Industry→Process Mapper）：行业→工艺模板→能流/物料流 → “工艺假设卡”（可人工校正）
基线能耗与负荷合成 Agent（Baseline Synthesizer）：计量/统计/估算 → 年量 + 典型日/8760 曲线（含区间）
基线碳核算 Agent（GHG Accountant）：S1/S2 核算、贡献拆分、口径声明 → 排放清单与不确定性
资源与气候校正 Agent（Resource & Climate）：HDD/CDD、出力、极端天气 → 资源上限与韧性约束
措施筛选 Agent（Measure Screener）：措施库按适用条件筛选 → 候选措施列表（适用性评分与补数项）
能源系统方案 Agent（Energy System Planner）：三联供/余热/热泵/储能/微网组合优化 → 装机/运行/成本/减排/可靠性
CO₂ 捕集与利用 Agent（CCU Planner）：来源→捕集路线→利用/封存约束与成本 → CCU/CCS 方案包与边际成本
副产品供需匹配 Agent（Byproduct Matcher）：副产品→潜在需求→距离/管网/合规评分 → 可实现 pairing 与可变现量
副产品定价与合同风险 Agent（Value & Risk）：价格波动/替代品/合同折价 → 风险调整后收益与现金流
财务整合 Agent（Finance Integrator）：组合现金流 → NPV/IRR/MACC、分期投资 → 组合经济性与敏感性
合规与政策 Agent（Policy & Compliance, RAG）：政策匹配、引用与适用性结论 → 合规风险点
报告编排 Agent（Report Orchestrator）：写入模板 → 报告包（PDF/Word/HTML）+ 结果落库

5) 架构层：UI 与整体技术接口
面向产品化：地图主视图 + 情景工作台 + 副产品网络 + 报告中心；后端以异步任务与可复算为核心。
可工程化
复制本节文本
地图主视图：画选区（polygon）、行政区检索、企业点聚合、图层管理（企业/行业热力/基础设施/资源/约束）
园区画像侧栏：产业结构、能耗/碳排基线、数据完备度、重点企业列表（可点选）
方案工作台（Scenario Studio）：假设输入（电价/气价/碳价/WACC/开工率/建设期）；方案开关（CCHP/热泵/光伏/储能/CCU/共生）；输出对比（KPI 表 + 雷达/帕累托/MACC）
副产品网络视图：供需网络图（企业—副产品—企业），查看纯化/运输/合同假设与现金流
报告中心：模板选择（政府版/技术版/投资版）、一键导出、附录下载（参数/引用/明细）
核心接口（示例）：面向“区域分析—情景—结果—报告”闭环。
POST /regions/analyze        # 输入 polygon + scenario → 返回 job_id
GET  /jobs/{job_id}          # 进度、当前 agent、日志、中间结果快照
GET  /regions/{region_id}/profile
POST /scenarios              # 创建/更新情景（价格、贴现率、政策口径）
GET  /results/{scenario_id}  # 全量指标与可视化数据
POST /reports/export         # 选择模板/语言 → 返回报告文件 id
POST /data/ingest            # 数据入库（POI/名录/价格/政策文档）
GET  /data/lineage           # 指标追溯（某个数字的来源与计算路径）
异步计算链：analyze → optimize → finance → report
每次运行写入：输入快照、参数快照、模型版本、输出快照
GIS：PostgreSQL + PostGIS；瓦片服务（可选）
计算服务：Python（工程模型/优化/财务）+ 求解器（LP/MILP/仿真）
多智能体编排：工具调用式 Agent Runtime + 统一中间结果 Schema
RAG：向量库 + 文档解析与引用机制
结果存储：关系库（指标落表）+ 对象存储（报告、图表、原始文件）
可观测性：任务日志、指标追溯、版本管理、权限审计
